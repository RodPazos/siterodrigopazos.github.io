Making of

Projeto Integrador I: Página Pessoal

O processo de desenvolvimento do site foi longo, iniciado por um Brainstormming, concepção da UX (experiência do usuário) e esboços de Wireframes para delimitar a UI (user interface) inicial.

Posteriormente, testes no computador foram feitos, digitalizando alguns Wireframes e inserindo a palheta de cores, alguns elementos gráficos e a navegação geral do site.

As seções foram definidas e conteúdo delimitado para cada uma, assim como preenchido e editado diversas vezes. Na sequencia, as imagens, videos e links foram reunidos para associarem a cada elemento.

A programação do site veio em seguida, iniciada pelo HTML e posteriormente estilizado pelo CSS.

Diversos testes foram feitos para ajustar o layout e os estilos para a UX e UI esperada, assim como no final foi reunido o material necessario para a criação dessa página extra, do Making Of.
